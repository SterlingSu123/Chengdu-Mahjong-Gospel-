## Why this exists / 为啥写这个

I’m not a theologian. I’m someone who sat in a Chengdu hotpot restaurant and realized that the Last Supper makes more sense if you play it as a mahjong game.

我不是学神学的。我只是在成都的火锅店里坐着坐着，突然觉得最后的晚餐——要是在麻将桌上摆一哈，好像更对味儿。

This is not a translation. It’s a retelling — in a language that tastes like red oil, sounds like tile clacks, and smells like a late-night hotpot.

这不是翻译，是重新摆一哈。用红油味的语言，麻将声的节奏，深夜火锅店的气息。
